# 🌱 Sugarcane Production Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on global sugarcane production data to understand production patterns across countries and continents.

---

## 📊 Tools & Libraries Used

- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔍 Project Objectives

- Analyze top sugarcane producing countries  
- Compare acreage and production  
- Perform correlation analysis  
- Visualize insights using bar charts and heatmaps  
- Practice real-world data cleaning and preprocessing  

---

## 📁 Project Files

- `EDA-Sugarcane-Project.ipynb` — Main analysis notebook  
- `List of Countries by Sugarcane Production.csv` — Dataset  
- `README.md` — Project documentation  

---

## 🧹 Data Cleaning

- Removed commas and formatting from numeric columns  
- Converted string values to numeric types  
- Handled missing values  
- Prepared data for visualization and correlation  

---

## 📈 Key Insights

- Identified top sugarcane producing countries  
- Observed relationships between acreage and production  
- Used correlation heatmaps to understand numerical dependencies  
- Created visualizations for clearer interpretation
